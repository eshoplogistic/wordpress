== Description ==

eShoplogistic — интеграционный сервис, позволяющий интегрировать на сайт сразу несколько служб доставки. На данный момент это: CDEK, Boxberry, Почта России, IML, Деловые Линии, ПЭК, DPD, GTD, Байкал Сервис. По запросу подключим другие службы.

Основные возможности:

- расчет доставки в корзине и карточке товара
- несколько служб доставки в одной интеграции
- настройка вашей собственной службы доставки
- гибкие правила корректировки стоимости и сроков доставки
- виджет для карточки товара с возможностью оформления заказа
- виджет для лендингов с работой в режиме корзины
- Вес и габариты товаров для расчета могут быть указаны в карточках товаров на сайте или прописаны в настройках eShopLogistic по умолчанию.

В личном кабинете вы самостоятельно выбираете какие службы доставки вам нужны. А так же настраиваете правила обработки тарифов.
Правила могут быть составлены в зависимости от:

- Способа оплаты
- Количества товаров в заказе
- Веса товаров в заказе
- Суммы заказа
- Способа доставки
- Направления доставки (конкретный город или область).

Корректировка может быть произведена относительно:

- Раcчётной стоимости доставки
- Суммы заказа
- Расчётной стоимости доставки + суммы заказа
- Указанной суммы
- Указанного срока.

eShopLogistic — не агрегатор. Мы не вмешиваемся в ваши взаимоотношения с курьерскими компаниями и не являемся посредниками в части передачи заказов в ЛК служб доставки. С нами не нужно заключать договор. Мы только делаем ваш сайт лучше, предоставляя удобный инструмент расчета доставок, для управления которым не нужно привлечение программиста.

Модуль распространятся бесплатно, оплата только за использование сервиса по тарифам eShoplogistic.

Для использования модуля необходимо зарегистрировать учетную запись на платформе eShoplogistic.

При необходимости наша поддержка окажет квалифицированную техническую помощь с подключением и настройками модуля.
Подробную документацию вы также можете найти на нашем сайте eShoplogistic

== Installation ==

1.После активации плагина в главном меню админ-панели появится пункт меню «WC eShopLogistic». Для настройки работы плагина перейдите в этот пункт меню.
2.Для начала вам нужно указать API ключ для сайта, который вы можете найти в своём личном аккаунте в системе eShopLogistic (см. документацию eShopLogistic).
3.В настройках вам нужно указать (ассоциировать) какой метод оплаты eShopLogistic соответствует доступным на данный момент методам оплаты Woocommerce.
